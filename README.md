# Eagle Eye — Tank Run

A small browser tank game. Top-down. Mouse + WASD.

Plays at `index.html` — no build step. Open the file or host it as static.

## Controls

- **WASD** — drive (tank rotates to face direction)
- **Mouse** — aim turret
- **Left Click** — cannon (slow, heavy, one-tap on basic enemies)
- **Right Click** — machine gun (750 RPM, 4 hits to kill)
- **Q** — class ability

## Tank classes

- **Light · Scout** — fast and fragile, dash ability
- **Medium · Striker** — balanced, smoke screen ability
- **MBT · Iron Wall** — slow and tough, reactive armor ability

## Maps

Desert · Snowfield · Jungle · Forest · Fulda Pass
